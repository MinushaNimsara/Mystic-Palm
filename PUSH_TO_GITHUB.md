# Push Mystic Palm to GitHub

Your code is committed locally but hasn't been pushed yet. Follow these steps:

## Step 1: Open Terminal in Cursor
- Press **Ctrl + `** (backtick) to open the terminal, or use **View → Terminal**

## Step 2: Run the push command
```powershell
git push -u origin main
```

## Step 3: When prompted
- **Username:** Your GitHub username (`MinushaNimsara`)
- **Password:** Use a **Personal Access Token**, NOT your GitHub password

### How to create a Personal Access Token
1. Go to https://github.com/settings/tokens
2. Click **Generate new token (classic)**
3. Name it (e.g. "Mystic Palm")
4. Check the **repo** scope
5. Click **Generate token**
6. Copy the token and paste it when git asks for "Password"

---

**Alternative: Use GitHub Desktop**
1. Download https://desktop.github.com/
2. Sign in with your GitHub account
3. File → Add Local Repository → select your `palm hand` folder
4. Click **Push origin**
